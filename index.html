Guides d'utilisation & Assistance technique au quotidien

Sur Mac (Avec TextEdit)

1. Préparer TextEdit (Crucial) :
Ouvrez un document vierge dans TextEdit. Allez tout en haut dans le menu Format > Convertir au format Texte Standard (Raccourci : Cmd+Maj+T).

2. Enregistrer :
Collez tout le code et enregistrez sous le nom cuisine.html. Validez l'utilisation de l'extension .html.

3. Modifier plus tard :
Pour coller une mise à jour d'une IA, faites un clic droit sur le fichier > Ouvrir avec > TextEdit.

Sur Windows (Avec Bloc-notes)

1. Créer le fichier :
Ouvrez le Bloc-notes (Notepad) et collez l'intégralité du code fourni par l'IA.

2. Enregistrer :
Allez dans Fichier > Enregistrer sous...
• Nommez le fichier : cuisine.html
• Type de fichier : Sélectionnez Tous les fichiers (*.*)
• IMPORTANT : Réglez l'encodage sur UTF-8 (accents corrects).

3. Modifier plus tard :
Faites un clic droit sur votre fichier > Ouvrir avec > Bloc-notes.

Sur Smartphone (Safari / Chrome)

🍎 Sur iPhone (Accès Safari direct via GitHub Pages) :
1. Créez un compte sur github.com.
2. Créez un dépôt public nommé cuisine.
3. Renommez votre fichier local en index.html et importez-le sur GitHub via "Add File" > "Upload Files". Validez.
4. Allez dans Settings > Pages. Sous "Branch", sélectionnez main et cliquez sur Save.
5. Votre URL de site s'affiche en haut de cette page après 1 min ! Ouvrez-la dans Safari, faites Partager > Sur l'écran d'accueil pour l'installer comme une vraie application.

🤖 Sur Android :
Dans votre gestionnaire de fichiers, appuyez longuement sur le fichier, sélectionnez Ouvrir avec... et choisissez Chrome.

Évolutions futures avec une IA

Pour faire évoluer votre application sans jamais perdre vos recettes ni votre stock au fil des mois :

  1. Dans l'onglet Configuration, cliquez sur le bouton "Exporter la sauvegarde (JSON)" pour télécharger vos données réelles de cuisine.
  2. Donnez ce fichier de sauvegarde ainsi que votre fichier .html à l'IA.
  3. Dites simplement à l'IA : "Voici mon application et mes données actuelles, peux-tu y ajouter la fonctionnalité suivante..."

L'IA pourra adapter le code de l'interface tout en préservant l'intégralité de vos réserves de nourriture !
